Codisimus.com

The first "project" I'm excited to share is actually this website. Web development is not typically my cup of tea, but I still like to practice it now and then. This website is fairly simple in that it is completely static. There is no backend that serves the pages, only the HTML/CSS/JS/etc files which are hosted on GitHub. I have decided to design my website using this approach to serve as a proof of concept that a server is not needed in some cases and free site hosting (such as from GitHub) may be sufficient. There are several "features" of these webpages, such as using CSS styling in place of JavaScript, on which I will provide more detail in a coming post.

I enjoy programming to make complicated tasks easier, automate tedious work, and create things that would otherwise be impossible without the help of a computer.
